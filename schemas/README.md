# Schemas

Versioned machine-readable JSON Schema documents for:

- the experiment **manifest** (JSON canonical; YAML accepted for authoring), and
- the **AUCX** container parts (see `docs/decisions/ADR-0003`).

These are added in later phases (manifests in Phase 3, AUCX in Phase 6). Each
schema is versioned so archives remain readable across format revisions.
